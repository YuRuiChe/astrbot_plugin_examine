## 📦 前置操作与安装
> **⚠️ 本插件仅适配QQ，以下步骤仅作参考，请以实际情况为准**

1. 在**QQ**新建一个审核群，记住主群与审核群的群号

2. 安装本插件

3. 在**Astrbot**中新建一个配置文件，更改以下配置
   - 关闭**AI配置-Agent 执行方式-启用**
   - 关闭**平台配置-基本-只 @ 机器人是否触发等待**
   - 开启**平台配置-基本-禁用自带指令**
   - 关闭**平台配置-白名单-启用白名单**
   - 在**插件配置-插件-可用插件**中，点击**选择插件集合**，点击**自定义选择**，只勾选**astrbot_plugin_examine**，若勾选了其他插件可能引发一些未知问题
   
   记得保存

4. 在**Astrbot**中新建一个bot，并接入任意消息平台（要求支持QQ，此处推荐Napcat）

## ⚙️ 插件配置
在本插件的配置文件中根据实际情况进行配置\
   题库格式示例:\
   **题目.txt**
   ```
   北国风光的下一句是什么
   明月几时有的下一句是什么
   寒雨连江夜入吴的下一句是什么
   ```
   **选项.txt**
   ```
   a.千里冰封[)b.平明送客楚山孤[)c.把酒问青天
   a.平明送客楚山孤[)b.千里冰封[)c.把酒问青天
   a.把酒问青天[)b.平明送客楚山孤[)c.千里冰封
   ```
   **答案.txt**
   ```
   a
   c
   b
   ```
   请确保**同一道题**的题目、选项、答案处于三个文件的**同一行**，然后移动文件于**data**文件夹下\
   若开启了随机抽题，则会从题库中**随机抽出**题目；若关闭，则按照题目的**先后顺序**排列。最终输出的题目数量为**目标题数**
