# Examine（入群自动考核）：简简单单的入群自动考核插件

> 我们会用心去做好每一个功能，欢迎提交 Issue

---

## ✨ 简介
- 能实现**入群考核**的功能
- （大概）能**替代**你的调查问卷
- 建议将bot**设为管理员**

## 📦 前置操作与安装
> **⚠️ 本插件仅适配 QQ ，以下步骤仅作参考，请以实际情况为准**

1. 在 **QQ** 新建一个审核群，记住主群与审核群的群号

2. 使用你原有的配置文件，或在 **Astrbot** 中新建一个配置文件，更改以下配置（皆为非必须，可以什么都不改）：
   - 关闭**平台配置-基本-只 @ 机器人是否触发等待**
   - 开启**平台配置-基本-禁用自带指令**
   - 在**插件配置-插件-可用插件**中，点击**选择插件集合**，点击**自定义选择**，只勾选 **astrbot_plugin_examine** ，若勾选了其他插件可能引发一些未知问题，例如指令冲突

3. 使用你原有的bot，或在 **Astrbot** 中新建一个bot并接入任意消息平台（要求支持 QQ，此处推荐 Napcat）

---

## ⚙️ 插件配置
在本插件的配置文件中根据实际情况进行配置

题库格式示例:\
**question_bank.json**
```json
{
   "1": {
      "question": "北国风光的下一句是什么",
      "option": "a.千里冰封/nb.平明送客楚山孤/nc.把酒问青天",
      "answer": "a"
  },
   "2": {
      "question": "明月几时有的下一句是什么",
      "option": "a.平明送客楚山孤/nb.千里冰封/nc.把酒问青天",
      "answer": "c"
   },
   "3": {
      "question": "寒雨连江夜入吴的下一句是什么",
      "option": "a.把酒问青天/nb.平明送客楚山孤/nc.千里冰封",
      "answer": "b"
   }
}
```
请确保题库是按照格式书写，然后移动文件于 **/data** 文件夹下，推荐放置于 **data/plugin_data/astrbot_plugin_examine** 目录下\
若开启了随机抽题，则会从题库中**随机抽出**题目；若关闭，则按照题目的**先后顺序**排列
